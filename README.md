<h1 align="center">🍰 Kanha Bakers</h1>

<p align="center">
  A modern <b>multi-page bakery website</b> built with <b>Next.js, TypeScript, and Tailwind CSS</b>.
</p>

<hr/>

<h2>🚀 Features</h2>
<ul>
  <li>🏠 Multi-page layout (Home, Menu, About, Contact, Cart, Checkout)</li>
  <li>🍩 Product listing with reusable components</li>
  <li>🛒 Cart system using Context API</li>
  <li>⚡ Fast performance with Next.js App Router</li>
  <li>🎨 Responsive UI using Tailwind CSS</li>
  <li>📱 Mobile-friendly design</li>
</ul>

<hr/>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li><b>Frontend:</b> Next.js</li>
  <li><b>Language:</b> TypeScript</li>
  <li><b>Styling:</b> Tailwind CSS</li>
  <li><b>State Management:</b> React Context API</li>
</ul>

<hr/>

<h2>⚙️ Installation & Setup</h2>

<ol>
  <li><b>Clone the repository</b></li>
</ol>

<pre>git clone https://github.com/Kannu20/Kanha-Cafe-.git</pre>

<ol start="2">
  <li><b>Navigate to project</b></li>
</ol>

<pre>cd Kanha-Bakers</pre>

<ol start="3">
  <li><b>Install dependencies</b></li>
</ol>

<pre>npm install</pre>

<ol start="4">
  <li><b>Run development server</b></li>
</ol>

<pre>npm run dev</pre>

<p><b>Open in browser:</b> http://localhost:3000</p>
